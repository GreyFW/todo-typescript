# Todo App (Vue 3 + TypeScript)

Лабораторная работа №8. Перенос Vue-проекта на TypeScript.

## Стек технологий

- Vue 3 (Composition API, script setup)
- TypeScript
- Pinia (State Management)
- Vue Router
- ESLint + Prettier

## Установка и запуск

1. **Установка зависимостей:**
   - `npm install`
2. **Запуск в режиме разработки:**
   - `npm run dev`
3. **Сборка проекта (Production):**
   - `npm run build`
4. **Проверка типов и линтинг:**
   - `npm run type-check`
   - `npm run lint`

## Функционал

- Регистрация и авторизация пользователей (Mock-данные).
- Создание, удаление, редактирование задач.
- Фильтрация задач (выполненные / текущие).
- Таймер для задач.
